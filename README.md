# AI驱动的机器学习教学闭环平台


## 项目简介

本平台专为高职《人工智能导论》《机器学习基础》课程设计，是从"AI辅助实验 → AI智能判题 → AI教学诊断 → 精准教学决策"的完整**四步教学闭环平台**。学生通过交互式可视化页面动手探索算法原理，AI自动批改编程实训并追踪错题，教师看板汇聚全班学情数据，辅助精准定位教学薄弱点。

## 核心亮点：四步教学闭环

| 步骤 | 环节 | 说明 |
|------|------|------|
| 1 | **AI辅助实验** | 11个交互式可视化页面，零代码调节参数，实时观察算法行为 |
| 2 | **AI智能判题** | DeepSeek API 驱动的编程实训考试模式，自动评判代码正确性，追踪错题知识点 |
| 3 | **AI教学诊断** | 教师看板汇总学生答题数据，自动生成不及格知识点分布与教学效果证据 |
| 4 | **精准教学决策** | 基于真实学情数据，一键导出Excel报告，指导针对性教学干预 |

## 功能页面（10个）

### 首页
四步教学闭环流程展示，12大功能入口快速导航。

### 数据探索
数据集统计表、特征分布直方图、相关性热力图，支持鸢尾花/手写数字/葡萄酒等多数据集切换。

### 线性回归
梯度下降过程可视化，残差诊断图，三种学习率（小/中/大）动态收敛对比。

### KNN分类
决策边界随K值实时变化，不同距离度量（欧氏/曼哈顿）准确率对比。

### K-Means聚类
聚类中心迭代过程可视化，肘部法则图自动判定最佳K值。

### 模型对比
四维雷达图（准确率/精确率/召回率/F1）+ 详细指标对比表，直观评估算法优劣。

### 神经网络
逐层激活值直方图，梯度消失/梯度爆炸可视化对比，学习率与Adam优化器效果对比。

### CNN卷积神经网络
卷积核滑动 → 特征图提取 → 池化降维 → 全连接分类，四步完整可视化流程。

### 算法知识库
覆盖线性回归/KNN/K-Means/神经网络/CNN等算法，包含原理讲解、应用场景、常见FAQ与代码示例。

### AI编程实训
基于DeepSeek API的智能编程评测系统：考试模式、错题自动追踪、AI变体出题、个性化学习建议。

### 教师看板
5项核心指标卡片（平均分/及格率/优秀率/提交次数/薄弱知识点），教学效果证据一览，AI诊断建议，支持Excel一键导出。

## 快速开始

### 方式1：Hugging Face Spaces（推荐）

国内直连，免费部署，无需服务器：

1. 将项目推送到 Hugging Face Space（选择 Streamlit 模板）
2. 在 Space Settings 中添加环境变量 `DEEPSEEK_API_KEY`
3. Space 自动安装依赖并启动，获得公网访问链接

### 方式2：本地部署

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动应用
streamlit run Home.py

# 3. 浏览器访问 http://localhost:8501
```

环境变量配置（二选一）：
- 创建 `.streamlit/secrets.toml`，添加 `DEEPSEEK_API_KEY = "your-key"`
- 或在系统环境变量中设置 `DEEPSEEK_API_KEY`

## 题库说明

内置 **15道编程实训题**，覆盖全部7个知识点：

| 知识点 | 题数 | 难度分布 |
|--------|------|----------|
| 线性回归 | 3 | 简单×1、中等×2 |
| KNN分类 | 2 | 简单×1、中等×1 |
| K-Means聚类 | 2 | 中等×2 |
| 神经网络 | 3 | 中等×2、困难×1 |
| CNN卷积神经网络 | 2 | 中等×1、困难×1 |
| 模型对比 | 1 | 中等×1 |
| 综合 | 2 | 困难×2 |

每道题包含：题目主题、详细描述、代码框架模板、参考答案、难度等级。AI判题基于DeepSeek API自动评估学生代码与参考答案的语义一致性，支持错题知识点追踪与变体出题。

## 教学价值

### 解决的核心痛点
1. **原理抽象**：通过逐层可视化（如CNN卷积核滑动→特征图→池化→分类）直观呈现算法内在机制
2. **调参黑箱**：实时滑块调节超参数，立即观察决策边界/损失曲线/准确率变化
3. **缺乏反馈闭环**：AI自动判题→错题追踪→教师看板→精准干预，形成完整教学数据链
4. **教学决策凭经验**：教师看板基于真实学生答题数据，提供AI诊断建议与可导出报告

### 适用场景
- 课堂演示：教师实时展示算法行为，替代静态PPT
- 课后实训：学生自主完成AI编程实训，AI即时反馈
- 教学诊断：教师通过看板快速定位班级薄弱知识点
- 课程设计：作为机器学习入门综合实践项目
