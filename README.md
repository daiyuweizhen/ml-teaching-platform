# 机器学习算法可视化工坊

> 基于Streamlit的交互式教学平台

## 🎯 项目概述

本平台专为高职《人工智能导论》《机器学习基础》课程设计，通过**交互式可视化**方式，帮助学生理解机器学习核心算法的工作原理。

### ✨ 核心特点
- **零代码交互**：无需编写代码，通过滑块、按钮即可探索算法
- **实时可视化**：调节参数后立即呈现图表变化
- **多算法集成**：线性回归、KNN分类、K-Means聚类一站式体验
- **数据探索**：内置多个经典数据集，支持特征分析
- **模型对比**：同一界面对比不同算法性能

## 🚀 快速开始

### 环境要求
- Python 3.8+
- 依赖包：见 `requirements.txt`

### 安装步骤

1. 克隆或下载项目
```bash
git clone <repository-url>
cd ml_visualization_platform
```

2. 安装依赖
```bash
pip install -r requirements.txt
```

3. 运行应用
```bash
streamlit run Home.py
```

4. 打开浏览器访问 `http://localhost:8501`

## 🧭 学习路径指南

### 📚 推荐学习顺序（从基础到高级）

1. **数据探索** (01_Data_Explorer_数据探索.py)
   - 理解数据集特性
   - 掌握特征分析方法
   - 学习数据预处理

2. **线性回归** (02_Linear_Regression_线性回归.py)
   - 理解监督学习基础
   - 掌握梯度下降原理
   - 学习参数优化方法

3. **KNN分类** (03_KNN_Classifier_KNN分类.py)
   - 理解基于实例的学习
   - 掌握距离度量概念
   - 学习分类算法应用

4. **K-Means聚类** (04_KMeans_Clustering_K均值聚类.py)
   - 理解无监督学习
   - 掌握聚类分析原理
   - 学习肘部法则应用

5. **模型对比** (05_Model_Comparison_模型对比.py)
   - 综合比较不同算法
   - 理解算法选择原则
   - 学习性能评估方法

6. **神经网络** (06_Neural_Network_神经网络.py)
   - 理解深度学习基础
   - 掌握前向/反向传播
   - 学习网络结构设计

7. **算法知识** (07_Algorithm_Knowledge_算法知识.py)
   - 理论知识总结
   - 算法原理深入理解
   - 应用场景分析

## 🔧 技术架构

- **前端/后端**：Streamlit（自动生成Web界面）
- **机器学习**：scikit-learn（算法模型与数据集）
- **可视化**：Plotly（动态交互式图表）
- **数据处理**：Pandas、NumPy
- **AI辅助开发**：DeepSeek大模型

## 🎯 教学价值

### 解决的教学痛点
1. **原理抽象**：通过可视化直观理解算法工作原理
2. **调参黑箱**：实时观察参数变化对模型的影响
3. **缺乏反馈**：立即看到调节参数后的效果变化

### 适合的教学场景
- 课堂演示：教师实时展示算法行为
- 课后练习：学生自主探索参数影响
- 实验报告：生成可视化结果用于分析
- 课程设计：作为机器学习入门实践项目
